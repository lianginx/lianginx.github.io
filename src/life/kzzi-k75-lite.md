# 珂芝（Kzzi）K75 Lite 说明书

[下载官方说明书](https://en.kzzi.com/media/files/20240705/20240705164922_9738.pdf)

## 电量指示灯

按下 `Fn + Enter` 查看电量，通过数字区 `1~0` 按键背光显示剩余电量。

| 指示灯状态       | 说明   |
| ---------------- | ------ |
| 红灯闪烁         | 电量低 |
| 常亮（接入电源） | 充电中 |
| 熄灭（接入电源） | 已充满 |

## 蓝牙连接

将右上角开关拨动到中间位置，表示蓝牙连接模式。

短按 `Fn + Q / W / E` —— 切换蓝牙频道

长按 `Fn + Q / W / E` —— 进入配对模式

## 键盘背光

| 组合键 | 说明         | 组合键 | 说明         |
| ------ | ------------ | ------ | ------------ |
| Fn + \ | 切换背光效果 | Fn + ↑ | 提高背光亮度 |
| Fn + , | 切换背光颜色 | Fn + ↓ | 降低背光亮度 |

## 切换修饰键

按下 `Fn + A` —— 切换至 Win 系统

按下 `Fn + S` —— 切换至 Mac 系统

## 功能键

| 组合键      | 说明  | 组合键      | 说明   |
| ----------- | ----- | ----------- | ------ |
| `Fn + HOME` | Pause | `Fn + END`  | Insert |
| `Fn + DEL`  | ScrLK | `Fn + PGUP` | PrtSc  |
