# 快速查询 Mac 磁盘信息

> 参考：[MacBook M1 查询硬盘写入教程 | 知乎](https://zhuanlan.zhihu.com/p/354385629)

## 安装磁盘工具

[smartmontools](https://www.smartmontools.org/) 是一款用于监控和分析硬盘健康状态的命令行工具。

通过 [Homebrew](./homebrew-guide.md) 安装：

```bash
brew install smartmontools
```

## 查询磁盘信息

安装后，即可查询磁盘信息：

```bash
smartctl -a disk0
```
